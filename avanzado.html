Principios Avanzados de la POO

Los principios SOLID permiten escribir código más estructurado, mantenible y escalable.

1. Principio de Responsabilidad Única (SRP)

Este principio establece que una clase debe tener una única razón para cambiar. Es decir, cada clase debe encargarse de una sola responsabilidad.

class Reporte:
    def generar(self):
        return "Generando reporte..."

class Guardado:
    def guardar(self, contenido):
        print(f"Guardando: {contenido}")

# Uso
reporte = Reporte().generar()
Guardado().guardar(reporte)

2. Principio de Abierto/Cerrado (OCP)

Una clase debe estar abierta para extensión, pero cerrada para modificación. Esto se logra mediante la herencia y el uso de interfaces.

class Calculadora:
    def operar(self, a, b):
        pass

class Suma(Calculadora):
    def operar(self, a, b):
        return a + b

class Resta(Calculadora):
    def operar(self, a, b):
        return a - b

operacion = Suma()
print(operacion.operar(5, 3))

3. Principio de Sustitución de Liskov (LSP)

Una subclase debe poder reemplazar a su clase base sin alterar el funcionamiento del programa.

class Ave:
    def volar(self):
        return "Volando"

class Pato(Ave):
    def volar(self):
        return "El pato está volando"

ave = Pato()
print(ave.volar())

4. Principio de Segregación de Interfaces (ISP)

Una clase no debe verse obligada a depender de métodos que no utiliza.

class AveVoladora:
    def volar(self):
        pass

class AveNadadora:
    def nadar(self):
        pass

class Aguila(AveVoladora):
    def volar(self):
        return "El águila vuela"

class Pinguino(AveNadadora):
    def nadar(self):
        return "El pingüino nada"

5. Principio de Inversión de Dependencias (DIP)

Las clases deben depender de abstracciones y no de implementaciones concretas.

class BaseDatos:
    def guardar(self, datos):
        pass

class MySQL(BaseDatos):
    def guardar(self, datos):
        print(f"Guardando en MySQL: {datos}")

class Aplicacion:
    def __init__(self, db: BaseDatos):
        self.db = db

    def procesar(self, datos):
        self.db.guardar(datos)

db = MySQL()
app = Aplicacion(db)
app.procesar("Información importante")
